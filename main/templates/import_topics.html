CSV File Format

The CSV file should contain the following columns:

Required Topic Fields:
  • topicName - Name of the topic
  • topicDescription - Description of the topic
  • topicNote - Notes for the topic
Optional Exercise Fields:
  • exerciseName - Name of the exercise
  • exerciseDescription - Description of the exercise
Optional Question Fields:
  • questionType - Type of question (must be one of: fill_blank, click_drag, matching, ordering, translating, multiple_choice, true_false)
  • questionPrompt - The question text
  • correctAnswer - The correct answer

Note: The first row should contain column headers. To include exercises and questions, all respective fields must be present.
